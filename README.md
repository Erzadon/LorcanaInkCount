# Lorcana Ink Curve Calculator

A React-based calculator for analyzing ink curves in Lorcana decks.

## Prerequisites

- **Node.js** 18 or later
- **npm** (comes with Node.js) or **Yarn**

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Run the test suite:

```bash
npm test
```

Create a production build:

```bash
npm build
```
